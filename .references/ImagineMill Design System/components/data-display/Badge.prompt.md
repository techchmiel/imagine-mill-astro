One-line: A compact uppercase status pill — use for content types ("VIDEO", "GUIDE"), statuses, and small counts.

```jsx
<Badge variant="info">New</Badge>
<Badge variant="success" dot>Published</Badge>
<Badge variant="navy">Video</Badge>
```

Variants: `neutral`, `info`, `success`, `danger`, `warn`, `navy`, `outline`. Set `dot` for a leading status dot.
